🎮 Maple2D - WinAPI 기반 2D 게임 서버 프로젝트
📌 프로젝트 개요
목표: WinAPI 기반의 2D 게임 클라이언트를 제작하고, IOCP 기반 멀티스레드 서버와 통신하는 구조의 게임을 구현합니다. 메이플스토리와 유사한 구조의 간단한 RPG 스타일의 게임을 목표로 합니다.

--------------------------------------

개발 환경

Language: C++

API: WinAPI

Server: IOCP 기반 멀티스레드 서버

Client: WinAPI 기반 2D 게임 렌더링
